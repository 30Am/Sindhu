# Sindhu Biswal — Growth Audit Website

A personal branding and services website for Sindhu Biswal, built with Next.js and Tailwind CSS.

## Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Database:** Supabase
- **Deployment:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

```
app/          → Pages (App Router)
components/   → UI components
lib/          → Supabase client
public/       → Static assets (logos, carousel images)
```
